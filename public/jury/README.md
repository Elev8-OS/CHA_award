# Jury Photos

Drop jury member portraits here. Naming convention:

- `prima.jpg` — Pak Prima Hartawan (also used for endorsement section)
- `reto.jpg` — Reto Wyss
- `maya.jpg` — Maya Susanti (when ready)
- `florian.jpg` — Florian Holm (when ready)

## Recommendations

- Square format, 400×400 to 800×800 pixels
- JPG (smaller file) or PNG/WebP
- Headshot-style: face centered, looks at camera
- Keep file size < 200KB for fast loading

If a photo is missing, the page will fall back to a colored initial-avatar.
